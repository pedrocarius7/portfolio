Sobre mim

Sou estudante de Tecnologia da Informação na FAETERJ e apaixonado por desenvolvimento de software. Minha trajetória inclui a participação no curso Residência de Software do Serratec em 2022.2, onde adquiri uma base sólida em programação. Já tive experiências práticas com Java e JavaScript, mas minha verdadeira paixão está no desenvolvimento front-end, usando tecnologias como HTML, CSS e JavaScript para criar interfaces modernas e funcionais.

Sou uma pessoa extrovertida e adoro trabalhar em equipe, pois acredito que a colaboração é essencial para o aprendizado e para alcançar resultados incríveis. Meu objetivo é continuar me especializando no front-end, sempre buscando inovar e oferecer experiências incríveis para os usuários.
